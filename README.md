## Project Overview
This is the backend of a single-page application (SPA) built using Node.js, Express.js, and MongoDB. It handles product data management, including APIs for searching, filtering, categorizing, and sorting products. It also supports pagination and integrates with Firebase for authentication.

## Features
- **RESTful API:** Provides endpoints for managing products, including search, filtering, categorization, sorting, and pagination.
- **MongoDB Integration:** Data is stored and managed using MongoDB.
- **Authentication:** Firebase is used for Google and Email/Password authentication.
- **Pagination:** Server-side pagination for efficient data loading.

## Technologies Used
- Node.js
- Express.js
- MongoDB
- Firebase Admin SDK (for authentication)
